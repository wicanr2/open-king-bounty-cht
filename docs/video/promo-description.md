## 完整版簡介(YouTube / 部落格)

1990 年的戰棋始祖《King's Bounty》(《魔法門》系列的前身),現在全程繁體中文,在 Windows、macOS、Linux 上免費就能玩。

這是以開源 openkb 引擎為基礎的繁體中文化專案。我們做的事:

- **全文中文化**:介面、兵種、法術、城鎮、劇情與結局字幕全部翻譯;職業、兵種、寶物的譯名照 1990 年代官方手冊走,不自己掰。
- **讓引擎開口說中文**:把舊的 SDL 1.2 引擎移植到 SDL2,加上 24×24 點陣中文渲染與高解析畫布,中文銳利不糊、不破版。
- **四套美術一鍵切換**:逆向破解 DOS、Mega Drive、Amiga 三個版本的原始資料,連同自由授權的 free 美術,遊戲中按 F8 即時切換;按 F9 開關 FM-Towns 背景音樂。
- **跨平台打包**:Windows、macOS(原生真 SDL2)、Linux AppImage,另有 Android 觸控版實驗中。
- **修掉一串玩家回報的跨平台問題**:首次啟動閃退、Windows/macOS 讀不到資料檔、選角後閃退、戰鬥畫面文字殘留、macOS「載入 SDL 失敗」等——每個都從 Wine/截圖逐一定位根因再修正,技術記錄(postmortem)公開在 repo。
- **免費玩,不需原版**:公開版用自由授權美術,不必自備原版遊戲就能玩到完整、中文化的《御封戰將》;若你擁有正版,完整版會顯示原版美術、原版開場(New World Computing 商標 + 原版標題)與原版配樂。
